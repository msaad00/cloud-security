# cloud-security

Cloud security skills collection — reusable security automation for AI agents.

## Structure

```
skills/
  <skill-name>/
    SKILL.md          # Skill definition (OpenClaw format)
    examples/         # Usage examples
```

## Usage

Each skill in `skills/` is a standalone security automation that can be invoked by AI agents (Claude Code, Cursor, etc.) via the OpenClaw skill format.
